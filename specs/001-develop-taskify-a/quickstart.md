# Quickstart: Taskify MVP

## Prerequisites
- .NET 8+ SDK
- PostgreSQL 15+

## Steps
1. Clone the repository and checkout the feature branch.
2. Ensure PostgreSQL is running and accessible.
3. Run database migrations (see contracts/ for schema).
4. Start the backend (Aspire orchestrator).
5. Start the frontend (Blazor Server).
6. Open the app in your browser and select a user to begin.

## Test Scenarios
- Move tasks between Kanban columns.
- Assign tasks to users.
- Add, edit, and delete comments on tasks.
- Verify real-time updates and task highlighting.
